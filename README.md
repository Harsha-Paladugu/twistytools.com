# twistytools.com

Hub for the TwistyTools puzzle sites:

- [pyraminx.twistytools.com](https://pyraminx.twistytools.com) (currently [pyraminx.net](https://pyraminx.net))
- [skewb.twistytools.com](https://skewb.twistytools.com) (currently [skewbiks.com](https://skewbiks.com))
- [fto.twistytools.com](https://fto.twistytools.com)

This repo hosts the twistytools.com landing page and is the single home of the shared Firebase configuration: `firebase.json`, `firestore.rules`, the rules tests, and migration scripts. The individual puzzle sites live in their own repos and deploy to subdomains via GitHub Pages.

See `TWISTYTOOLS-MIGRATION.md` for the consolidation plan and current status.
